Baseline hardening lintas OS dan cloud tanpa langsung mengubah produksi.

Kapabilitas

Dibuat untuk audit yang aman, evidence yang jelas, dan perubahan yang bisa ditinjau.

Cloud posture baseline

AWS, Azure, dan GCP diaudit untuk storage exposure, logging, monitoring, network exposure, dan kontrol security service prioritas tinggi.

Controlled remediation

AWS S3 mendukung apply terbatas untuk Public Access Block, default encryption, dan versioning dengan change manifest untuk bukti perubahan.

Rollback evidence

Rollback AWS memulihkan konfigurasi reversible dari nilai sebelum perubahan. Operasi yang tidak sepenuhnya reversible ditandai manual.

OS hardening scripts

Linux, macOS, dan Windows memiliki mode audit, plan, dan apply untuk baseline firewall, SSH, patching, audit policy, dan kontrol akses umum.

Policy overrides

File YAML tervalidasi mendukung override port administratif dan resource exclusions yang tetap muncul sebagai finding SKIP.

CI and release ready

Ruff, mypy, pytest coverage, ShellCheck, CodeQL, dependency audit, SBOM, checksums, dan provenance attestation dijalankan dalam pipeline.

Workflow

Mulai dari audit, review rencana, lalu apply hanya bila sudah siap.

Pendekatan default audit-only mengurangi risiko perubahan tidak disengaja. Tim dapat menyimpan report, mengulas rencana, dan menjalankan apply dengan --yes hanya untuk remediation yang deterministik.

  1. 01

    Audit

    Inventarisasi posture dan ekspor hasil ke text, JSON, atau SARIF.

  2. 02

    Plan

    Buat manifest before/after tanpa memanggil API mutasi.

  3. 03

    Apply

    Jalankan perubahan terbatas dengan change manifest sebagai bukti.

  4. 04

    Rollback

    Pulihkan setting reversible dan tandai batasan manual secara eksplisit.

Cakupan

Kontrol praktis untuk area risiko yang sering muncul.

AWS

S3 public access, encryption, versioning, CloudTrail, Config, GuardDuty, Security Hub, KMS, default security groups, Organizations.

Azure

Storage TLS, storage network rules, NSG administrative exposure, Defender plans, subscription activity log export.

GCP

Cloud Storage public IAM, uniform bucket access, firewall exposure, logging sinks, Security Command Center manual posture.

OS

Linux firewall/SSH/patching, macOS firewall/Gatekeeper/FileVault guidance, Windows firewall/SMB/auditing.

Mulai

Pasang provider yang dibutuhkan, lalu jalankan audit pertama.
