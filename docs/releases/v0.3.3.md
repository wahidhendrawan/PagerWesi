# PagerWesi v0.3.3

- Full English GitHub Pages landing page.
- Rollback support for EBS encryption by default, tool-created IAM Access Analyzers, and VPC Flow

- SARIF rule metadata with control help URIs.
- GHCR container image publishing from the release workflow.
- GitHub Actions Python setup upgraded to `actions/setup-python@v6`.
