# PagerWesi v0.9.0

PagerWesi v0.9.0 is the project rebrand release. The release keeps the existing
audit-first hardening scope while aligning
repository names, package metadata, CLI examples, GitHub Pages links, and container names under the
PagerWesi identity.

## Migration Notes

- Use the `pagerwesi` console command for new installs and examples.
- Pull container images from `ghcr.io/wahidhendrawan/pagerwesi`.
- Update documentation links to `https://wahidhendrawan.github.io/PagerWesi/`.
- Rename custom organization roles or session labels that still use previous branding.

## Container

```bash
docker pull ghcr.io/wahidhendrawan/pagerwesi:v0.9.0
docker run --rm \
  -e AWS_REGION=us-east-1 \
  ghcr.io/wahidhendrawan/pagerwesi:v0.9.0 aws --format json --output /dev/stdout
```
