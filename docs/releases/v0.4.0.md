# Automation Hardening v0.4.0

Released on 2026-06-17.

## Highlights

- AWS rollback now covers EBS encryption by default, IAM Access Analyzer, and VPC Flow Logs.
- Azure and GCP support non-mutating plan mode with plan manifest generation.
- Policy validation uses JSON Schema with precise error paths.
- SARIF output enriched with NIST CSF, ISO 27001, and CIS framework tags, full descriptions,
  help URIs, and security-severity scores.
- Control catalog and compliance mapping generated from a single-source registry.
- GHCR container image published on every release with provenance attestation.
- Node 20 warning eliminated (`actions/setup-python@v6`).
- Integration workflow triggers on pull requests for cloud/tests paths.

## Container

```bash
docker pull ghcr.io/wahidhendrawan/automation-hardening:v0.4.0
docker run --rm -e AWS_ACCESS_KEY_ID -e AWS_SECRET_ACCESS_KEY \
  ghcr.io/wahidhendrawan/automation-hardening:v0.4.0 aws --format text
```

