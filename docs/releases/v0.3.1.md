# PagerWesi v0.3.1

Released on 2026-06-17.

## Highlights

- Standalone quickstart guide.
- `pagerwesi policy validate --policy PATH`.
- AWS controls for EBS encryption by default, RDS encryption, VPC Flow Logs, and IAM Access
  Analyzer.
- Azure controls for Key Vault public access, SQL auditing, and diagnostic settings.
- GCP controls for service-account user-managed keys, KMS rotation, and audit logging.
- Optional LocalStack S3 integration harness.

Use [quickstart.md](../quickstart.md) for the latest setup instructions.
