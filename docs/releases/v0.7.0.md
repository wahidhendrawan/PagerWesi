# Automation Hardening v0.7.0

Released on 2026-06-17.

## Highlights

- Webhook notifications for Slack, Microsoft Teams, and PagerDuty with `--notify`.
- SOC 2 and PCI DSS evidence export with `--export-compliance`.
- Time-boxed exception and waiver handling with `--exceptions`.
- Docker CIS audit coverage for daemon TLS, user namespace remapping, host networking, and image
  tag pinning.
- Secrets scanner for AWS access keys, private keys, and hardcoded password assignments.
- Terraform plan scanner for unrestricted ingress, public S3 ACLs, wildcard IAM actions, and
  missing encryption configuration.
- Network/TLS scanner for TLS version, certificate validity, and unexpected open ports.
- Static HTML dashboard site generation with `--generate-dashboard`.
- Agent mode for periodic audit and drift alerting.

## Upgrade

```bash
pip install -e '.[aws,azure,gcp,k8s]'
```
