## Highlights

- GitHub Actions workflows opt into Node 24 execution.
- Optional scheduled/manual LocalStack integration workflow.
- Policy schema with AWS VPC Flow Logs apply settings.

- AWS organization aggregate controls for CloudTrail, Config, GuardDuty, and Security Hub.
- Azure subscription diagnostic settings use ARM REST lookup.
- Provider permission examples for AWS, Azure, and GCP.
